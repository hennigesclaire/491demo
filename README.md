# BMI Calculator 
## Introduction
Simple Java program that takes in height (inches and feet) and weight (lbs) and returns BMI. Includes text showing the BMI categories (underweight, normal, overweight, obese)
## Features
- UI: User interface designed with netbeans 
- Input Fields: Allows user input for height and weight
- BMI calculation: Returns BMI given user input when calculate button is pressed 
## Technologies 
Designed in netbeans using Java with Ant and Swing GUI form. Reference material for netbeans: [Netbean Tutorial](https://netbeans.apache.org/tutorial/main/kb/)
## How to run
Can be run using netbeans.
